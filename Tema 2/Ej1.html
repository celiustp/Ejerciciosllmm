HTML

Introduccion

HTML, siglas en ingles de HyperText Markup Language (‘lenguaje de marcas de hipertexto’), hace referencia al lenguaje de marcado para la elaboración de paginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de paginas web en sus diferentes versiones, define una estructura básica y un codigo (denominado codigo HTML) para la definicion de contenido de una pagina web, como texto, imagenes, videos, juegos, entre otros.

Primeras especificaciones
Tim Berners-Lee en 1991 describe 18 elementos que incluyen el diseño inicial y relativamente simple de HTML. Trece de estos elementos todavia existen en HTML 4.
